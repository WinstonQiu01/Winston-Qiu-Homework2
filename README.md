## Step 2: Business Workflow Definition
### Summarizing meetings into structured action items

### User
A project manager or operations coordinator who needs clear follow-up tasks after meetings.

### Input
Raw meeting notes, transcripts, or bullet points, plus optional context like project name, participants, and deadlines.

### Output
A structured summary with key discussion points, action items, owners, and deadlines, plus a short executive summary if needed.

### Value
This task is valuable because meeting notes are often messy and unstructured. Automating the summary saves time, reduces missed action items, and improves team coordination.
